# 📝 Sentiment Analysis Tool (BERT + Logistic Regression)

🚀 A Machine Learning project that classifies text into **Positive or Negative sentiment** using BERT embeddings and Logistic Regression.

---

## 📌 Features
- 🔍 Text sentiment classification  
- 🤖 BERT embeddings + Logistic Regression  
- 📊 Trained on Sentiment140 dataset  
- ⚡ Fast and efficient predictions  
- 🌐 Streamlit web app for real-time prediction  

---

## 🧠 Tech Stack
- Python 🐍  
- Transformers (BERT)  
- Scikit-learn  
- Pandas, NumPy  
- Streamlit
